== Description ==

Instrucciones para utilizar el plugin para Wordpress de Despegar.

En primera instancia usted debe obtener su “Script” generado por el sistema de Despegar, lo podrá obtener una ves que se registre como agencia en el sitio de Despegar. 
El siguiente enlace le pude servir de utilidad:
www.despegar.com.ar/comunidadafiliados/promoteQuimera/index

Una ves que tenga su script, instale el plugin en su plataforma Wordpress, de la manera habitual, puede optar por subir el plugin, o buscarlo dentro de la misma plataforma.

Finalizada la instalación del plugin proceda a activarlo y a ingresar a su panel de control (Una ves activado el plugin se mostrara en el menú izquierdo).

La primera ves que ingrese deberá copiar el Script como lo obtuvo de la pagina de Despegar.

A continuación proceda a personalizar a su gusto como se mostrara la sección del plugin dentro de su web.

Cuando termine de personalizar la sección, observara que dentro del mismo plugin, hay una pestaña llamada “Shortcode”, copie el shortcode que le ofrece, y péguelo en la sección de web donde se mostrara la caja de búsqueda del plugin.


== Installation ==


1. Cargar los archivos de plugin en el directorio '/wp-content/plugins/plugin-name', o instalar el plugin a través de la pantalla de plugins de WordPress directamente.
2. Activar el plugin a través de la pantalla 'plugins' en WordPress
3. Puede configurar el plugin en la pantalla 'Despegar' Box en Wordpress
